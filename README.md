<br></br>
This project, developed in Python using an IDE like PyCharm, is a Cyber Security Final Year Project focused on real-time network traffic analysis. It features a user-friendly GUI that acts as a packet sniffer, capturing live network data. The core functionality involves a Deep Learning model that analyzes the captured packet features to classify traffic as either "normal" or an "attack".

Network Intrusion Detection System with GUI
This project is a Python-based Network Intrusion Detection System (NIDS) with a user-friendly Graphical User Interface (GUI), developed using tkinter and scapy. 

- Select Network Interface: Easily choose an active network interface to begin sniffing packets.
- Real-time Packet Sniffing: Capture live network traffic, providing real-time insights into packet flow.
- Deep Learning-based Anomaly Detection: Integrate a pre-trained Deep Learning model (network_ids_model.h5) to analyze packet features and classify them as "normal" or "attack" in simple, understandable terms.
- Data Management: Export captured packets as PCAP files (raw packets) or JSON files (structured data with extracted features and predictions), and import existing PCAP files for offline analysis.
- Intuitive GUI: A minimalist and easy-to-navigate interface ensures that even users new to network analysis can quickly understand and utilize the system.

<br></br>

Key Features:
- Live packet capture and display.
- Real-time classification of network traffic (normal/attack).
- Protocol statistics and packet length monitoring.
- Flexible data export and import capabilities.
- Built with Python, tkinter, scapy, and TensorFlow.
<br></br>

Note: For optimal performance and to avoid conflicts, it's recommended to run the sniffer on wireless interfaces (e.g., Wi-Fi) or in environments where direct Ethernet cable connection won't work.
